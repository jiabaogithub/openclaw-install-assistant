# OpenClaw速装工具
助你在【Windows11 / Windows10】上，  
5分钟快速安装OpenClaw。  
而这原本可能要花上1~2小时。

## 速装工具使用方法
1. 鼠标右键点击【开始安装.cmd】文件，选择【以管理员身份运行】，即可开始安装。  
2. 安装成功后，会直接进入【初始设置(OpenClaw onboard)】环节。

## 安装环境重置使用方法
1. 鼠标右键点击【开始重置.cmd】文件，选择【以管理员身份运行】，即可开始重置。  
2. 重置后，会自动完全卸载openclaw、nodejs、git。

## 为什么不用OpenClaw官网的自动安装命令？
相比【OpenClaw官网的自动安装命令】，  
本程序不仅能自动安装OpenClaw需要的Nodejs环境，还解决了下面这些问题：  
(1)解决90%用户安装OpenClaw速度慢的问题  
(2)解决90%用户要手动安装Git的问题  
(3)解决90%的用户在安装OpenClaw时出现git使用ssh协议拉取依赖被拒绝的问题  
(4)解决90%的用户在安装OpenClaw时出现git拉取libsignal依赖包报错的问题   
(5)解决安装过程没有日志输出，导致不知道安装是否还在正常进行，只能干等的问题
